# 多人合伙投资份额分配系统

这是一个基于Python和SQLite数据库的多人合伙投资份额分配系统，可以将投资数据持久化存储到本地数据库。

## 功能特点

1. **多项目管理**：支持创建和管理多个投资项目
2. **合伙人管理**：添加、编辑、删除合伙人信息
3. **自动份额计算**：根据投资金额自动计算每个人的份额比例
4. **数据持久化**：所有数据存储在本地SQLite数据库中
5. **可视化展示**：图表展示份额分配比例
6. **操作日志**：记录所有操作历史

## 使用方法

### 直接运行

1. 确保已安装Python 3.6或更高版本
2. 将项目文件下载到本地
3. 运行以下命令启动应用：

```bash
python app.py
